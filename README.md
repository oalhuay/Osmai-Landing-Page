# Osmai Landing Page

Landing page para Osmai, un servicio de web design, diseño visual y software para pequeñas y medianas empresas.

## Stack

- Next.js 16
- React 19
- Lucide React
- CSS personalizado

## Comandos

```bash
npm install
npm run dev
npm run build
```

## Deploy manual en Vercel

1. Importar este repositorio desde Vercel.
2. Framework preset: `Next.js`.
3. Build command: `npm run build`.
4. Output: Vercel lo detecta automaticamente para Next.js.
